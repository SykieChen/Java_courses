# Java socket communicate

a simple java messaging practice project.  
## Usage:
Run server first, then client.  
- Client:
    ``` bash
    javac Client.java
    java Client
    ```
- Single thread server:
    ``` bash
    javac ServerSingle.java
    java ServerSingle
    ```
- Multi thread server:
    ``` bash
    javac ServerMulti.java
    java ServerMulti
    ```
